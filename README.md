# Math Trainer

An interactive web application for practicing math skills with adaptive difficulty and performance tracking.

## Features

- **Customizable Settings**: Configure the number of problems (up to 100) and maximum number range (2-9)
- **Real-time Feedback**: Immediate validation of answers with visual feedback
- **Performance Tracking**: Track errors and completion time
- **Polish Language Interface**: Fully localized UI in Polish
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## How to Use

1. Open `index.html` in your web browser
2. Configure your practice session:
   - Set the number of problems you want to solve
   - Choose the maximum number for multiplication problems
3. Click "Rozpocznij" (Start) to begin
4. Solve each multiplication problem by entering your answer
5. Review your results at the end, including:
   - Total problems solved
   - Number of errors
   - Time taken

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Installation

No installation required! Simply clone this repository and open `index.html` in your web browser:

```bash
git clone https://github.com/earkhipov/math-trainer.git
cd math-trainer
open index.html
```
